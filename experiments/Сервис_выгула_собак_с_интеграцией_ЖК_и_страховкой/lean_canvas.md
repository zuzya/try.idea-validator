## Solution
Создание платформы для выгула собак, которая будет интегрирована с экосистемами жилых комплексов, позволяя жильцам заказывать услуги выгула через мобильное приложение. В рамках сервиса будет предусмотрена страховка от Ингосстрах, а также возможность взаимодействия с местными питомниками и ветеринарными клиниками для предоставления дополнительных услуг. Платформа будет работать на базе Yandex Cloud и использовать СБП для оплаты.

## Target Audience
Владельцы собак, жители жилых комплексов, активные пользователи мобильных приложений

## Monetization
Комиссия с транзакции и подписка для владельцев собак
